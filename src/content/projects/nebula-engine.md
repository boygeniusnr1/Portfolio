# Nebula Engine

Nebula is a project dedicated to pushing the boundaries of **real-time graphics** on the web.

### Technical Stack:
*   **WebGPU** for low-level GPU access.
*   **TypeScript** for robust engine architecture.
*   **WGSL** for high-performance custom shaders.

### Goals:
The primary goal was to achieve cinematic lighting and physics simulations at 60fps within a standard browser environment.
