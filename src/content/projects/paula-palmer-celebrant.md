# Paula Palmer | Celebrant Website

A high-performance, accessible, and elegant website built for a professional celebrant. The project focuses on a premium "Glass-morphism" aesthetic combined with smooth, interactive user experiences.

### Technical Excellence
*   **Astro Framework:** Leveraging Astro for zero-JS footprints and near-instant load times.
*   **Custom Glassmorphism:** Implementation of modern mesh gradients and glass-like UI components using advanced CSS.
*   **Optimized Delivery:** Automated image processing with Sharp and seamless deployment via Vercel's edge network.

### Key Features
*   **Magnetic Interaction:** Custom JavaScript-driven cursor follower that provides a high-end, tactile feel to the navigation.
*   **Custom Lightbox Gallery:** A high-performance, gesture-capable photo gallery optimized for all devices.
*   **Serverless Communication:** Integrated Web3Forms for streamlined, backend-less inquiry handling.
*   **Built-in Analytics:** Full integration with Vercel Analytics for performance monitoring and user insights.

### Outcome
A sophisticated digital presence that mirrors the professionalism and warmth of Paula Palmer's celebrancy services, fully optimized for Core Web Vitals and user engagement.
